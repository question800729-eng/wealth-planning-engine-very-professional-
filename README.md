# Financial Planning Monte Carlo Simulator

A long-horizon (100-year) financial planning simulator using Monte Carlo methods.

## Features
- Inflation-adjusted expenses
- Income growth vs inflation separation
- Tax on realized gains only
- Jewellery as illiquid asset with liquidation penalty
- Bankruptcy & survival probability tracking
- Lognormal return modeling

## Run locally
```bash
pip install -r requirements.txt
python app.py
